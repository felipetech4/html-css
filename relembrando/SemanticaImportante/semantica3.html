Outras semânticas:

Tag "q"

Indica uma breve citação ou citação simples (geralmente na mesma linha)

Tag "blockquote"

Indica uma citação maior, com um deslocamento para a direita e normalmente com quebra de linha.
Também permite inserir o link de onde peguei a citação (esse link não fica visível para o usuário, é somente a semântica usada para facilitar pesquisas de buscadores.).
Para inserir o link basta usar o parâmetro cite= dentro da tag.
