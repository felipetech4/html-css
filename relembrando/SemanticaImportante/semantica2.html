Testanto mais algumas semânticas importantes

Tag code indica um texto com linguagem de programação SEM IDENTAÇÃO:

Usando a tag code:

document.getElementById('teste')

Sem a tag code:

document.getElementById('teste')

Tag pre code indica um texto com linguagem de programação SEM IDENTAÇÃO:

Usando a tag pre code:

 
                
                    num = int(input('Digite um número'))
                    if num % 2 == 0:
                        print(f'O número {num} é PAR)
                    else:
                        print(f'O número {num} é ÍMPAR)
                    print('Fim do programa')

Sem a tag pre code:

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é PAR) else: print(f'O número {num} é ÍMPAR) print('Fim do programa')
